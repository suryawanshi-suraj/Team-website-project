# Cloth Shop Website

This project is a simple static website for a clothing store, built using HTML and CSS. It includes a homepage showcasing featured clothing items with price tags.

## Structure

- `index.html` - The main landing page displaying products.
- `style.css` - Styles for layout, typography, and the product grid.

## Features

- Responsive grid layout for product listings
- Placeholder images that can be replaced with real photos
- Basic typography and color scheme

## Getting Started

To view the site, open `index.html` in a web browser.

## Customization

Feel free to:

1. Add more products by editing the HTML.
2. Swap placeholder images with actual product images.
3. Update prices and descriptions as needed.
